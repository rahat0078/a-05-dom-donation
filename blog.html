About JS DOM

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface which used to represent and interacts with the contents of HTML documents.

How do you select an element from the DOM?

To select an element from DOM some of these common methods are used :
1. getElementById: Select element using unique id.
2. getElementsByClassName: Select all elements with a class name.
3. getElementsByTagName: Selects all elements with tag name.
4. querySelector: Selects the first element that matches a CSS selector.
5. querySelectorAll: Selects all elements that match a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. This approach improves efficiency, reduces memory usage, and simplifies managing events, especially for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, some JavaScript methods like: 1. setAttribute() , 2. adding style : element.style
